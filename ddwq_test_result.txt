# Introduction
# coefficient : 指明密度的权重所占的百分比。
# coefficient = -1 表明密度与距离的权重直接相加用于计算（密度权重 + 体积权重/divisor）。
# coefficient > 0.0 则按密度与距离的权重百分比相加（距离权重×（1-coefficoent） + 密度权重×（coefficient）/ divisor)）。
# divisor : 密度的权重需要除以该数。（本身算出来的密度的权重太大， 因为体积太小）
#
